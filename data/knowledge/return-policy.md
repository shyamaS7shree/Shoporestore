# Return Policy

Customers can return eligible products within 7 days of delivery if the item is unused, unworn, undamaged, and has original tags and packaging.

Refunds usually take 5-7 business days after the returned product is received and checked.

Some products may not be returnable for hygiene or category reasons. Customers should check the product page and order details before raising a return request.

To request a return, customers should go to Profile, open My Orders, choose the order, and follow the return option if available.

If clothing, footwear, or size-based products do not fit, feel too tight, too loose, or the wrong size was received, customers can raise a return request from Profile > My Orders if the product is eligible.

Returned products should be unused, unworn, undamaged, and should include original tags, labels, accessories, and packaging.

If return or exchange is not visible in My Orders, customers should contact support through the Contact page with their order number, product name, size received, and size needed.
